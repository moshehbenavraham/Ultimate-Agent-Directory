## [0.3.0] - 2025-07-04

### Added
- Comprehensive AI agent ecosystem directory structure
- 10 major categories covering complete AI agent landscape
- Open-Source Agent Frameworks (LangChain, AutoGen, CrewAI, Semantic Kernel)
- No-Code/Low-Code Platforms (Flowise, Dify, Langflow, n8n, Activepieces)
- Research-Focused Frameworks (Google AI Co-Scientist, CAMEL, L2MAC)
- Learning Resources and Courses (DeepLearning.AI, Hugging Face, Coursera)
- GitHub Repositories and Communities (Awesome lists, Discord, Reddit)
- Specialized AI Agent Tools (Customer Service, Sales, Marketing, Data Analysis, Security, DevOps)
- Autonomous Agents (AutoGPT, BabyAGI, AgentGPT, SuperAGI)
- Browser Automation Agents (Browserbase, AgentQL, Playwright, Puppeteer)
- Coding Assistant Agents (GitHub Copilot, Cursor, Codeium, Aider)
- Enterprise Agent Platforms (Microsoft, Google, AWS, Salesforce, IBM, Oracle, SAP)

### Changed
- 247+ AI agents, frameworks, and platforms catalogued
- Detailed descriptions with links to official sources
- Categorized by use case and implementation type
